A responsive frontend web application built as part of the Flyrank Frontend Capstone Track.

## 🚀 Features (Planned)
- **Interactive UI:** Dynamic components styled with Tailwind CSS.
- **Type Safety:** Full TypeScript integration for robust development.
- **Modern Build Pipeline:** Fast HMR and bundling powered by Vite.

## 🛠️ Stack
- **Frontend:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## ⚙️ Setup & Commands

First, install the project dependencies:
```bash
npm install
```

### Available Scripts
* **Start development server:**
  ```bash
  npm run dev
  ```
* **Build for production:**
  ```bash
  npm run build
  ```
* **Run unit tests:**
  ```bash
  npm test
  ```
